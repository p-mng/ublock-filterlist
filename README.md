# ublock-filterlist

## Yet another filterlist for uBlock Origin

The goal of this filterlist is to block

- ads
- tracking scripts, elements etc.
- annoying cookie banners (e.g. fullscreen overlays that prevent you from scrolling and interacting with the website)
- content related to gambling/porn/etc. on "normal" pages

on various sites.

The list works best with all other filterlists provided by uBlock Origin by default + the EasyList Germany regional list.
